👄 完整發音規則 (Prononciation Complète)

📌 單母音 (Voyelles simples)

  • a, à, â → [a] 如：chat, là, pâte
  • e (開音節) → [ə] 如：le, petit
  • e (閉音節) → [ɛ] 如：mer, avec
  • é → [e] 如：été, café
  • è, ê, ë → [ɛ] 如：père, tête, Noël
  • i, î, ï, y → [i] 如：midi, île, maïs, stylo
  • o, ô → [o] 如：mot, hôtel
  • u, û → [y] 如：tu, sûr
  • ou, où → [u] 如：vous, où

🔔 複合母音 (Voyelles composées)

  • ai, ei → [ɛ] 如：mais, neige
  • au, eau → [o] 如：auto, beau
  • eu, œu → [ø]/[œ] 如：peu, cœur
  • oi → [wa] 如：moi, trois
  • ui → [ɥi] 如：lui, nuit

👃 鼻母音 (Voyelles nasales)

  • an, am, en, em → [ã] 如：dans, jambe, vent, temps
  • in, im, ain, aim, ein → [ɛ̃] 如：vin, impossible, pain, faim, plein
  • on, om → [ɔ̃] 如：bon, nom
  • un, um → [œ̃] 如：un, parfum
  • ⚠️ 注意：鼻母音後若接n/m則不鼻化，如：bonne [bɔn]

🎯 子音發音規則 (Consonnes)

C 的發音

  • c + a/o/u → [k] 如：café, encore, cube
  • c + e/i/y → [s] 如：cinéma, centre
  • ç → [s] 如：français, garçon

G 的發音

  • g + a/o/u → [g] 如：gare, gomme
  • g + e/i/y → [ʒ] 如：géographie, gilet
  • gu + e/i → [g] 如：guerre, guide

S 的發音

  • 字首或子音後 → [s] 如：sac, aussi
  • 兩個母音間 → [z] 如：rose, maison
  • ss → [s] 如：poisson, classe

其他重要子音

  • h → 不發音 (分h muet和h aspiré)
  • r → [ʁ] 小舌顫音 如：Paris
  • j → [ʒ] 如：jour, bonjour
  • w → [v] 如：wagon

🔤 子音組合

  • ch → [ʃ] 如：chat, chocolat
  • ph → [f] 如：téléphone, photo
  • gn → [ɲ] 如：espagnol, montagne
  • qu → [k] 如：question, quoi
  • th → [t] 如：thé, théâtre
  • ll → [l] 如：ville (例外：fille [j])
  • -ill- → [j] 如：famille, fille

📍 字尾發音規則

常見字尾

  • -er, -ez → [e] 如：parler, allez
  • -et → [ɛ] 如：et, secret
  • -tion → [sjɔ̃] 如：nation, attention
  • -sion → [zjɔ̃] 如：télévision
  • -eur → [œʁ] 如：professeur

不發音的字尾

  • -e, -es, -ent (動詞) 通常不發音
  • -s, -x, -z, -t, -d, -p 字尾通常不發音
  • 例外：CaReFuL 字尾 (c, r, f, l) 通常發音

🔗 連音規則

  • 必須連音：冠詞+名詞 (les enfants [lezɑ̃fɑ̃])
  • 必須連音：形容詞+名詞 (un petit ami)
  • 必須連音：代詞+動詞 (vous êtes [vuzɛt])
  • 禁止連音：et 之後 (et il)
  • 禁止連音：h aspiré 之前 (les / haricots)

❌ 特殊情況與例外

  • x → [ks] 如：taxi, boxe
  • x → [gz] 如：examen, exemple (ex+母音)
  • x → [s] 如：six, dix (數字6, 10)
  • y → [i] 當母音 如：stylo
  • y → [j] 當半母音 如：yaourt

📚 不定冠詞 Articles indéfinis (a/an)

在一個未被提及或定義的名詞前面使用 ⚠️ 法文名詞不能裸奔！

單數 複數
陽性 un des
陰性 une
J'ai un stylo. 我有一支筆。(stylo 陽性)
J'ai une voiture. 我有一輛車。(voiture 陰性)
J'ai des vélos. 我有一些腳踏車。(複數)
⚠️ 否定句變化：un/une/des → de
J'ai une voiture. → Je n'ai pas de voiture. (我沒有車)
J'ai des vélos. → Je n'ai pas de vélo. (我沒有腳踏車)
💡 特例：c'est 句型保留 un/une
C'est un stylo. → Ce n'est pas un stylo. (這不是一支筆)

📖 定冠詞 Articles définis (the)

用於特指、第二次提及、固定含義（學科/語言/國家/獨一無二的事物）

單數 母音開頭單數 複數
陽性 le l' les
陰性 la
J'ai un chien, le chien a deux ans. 我有一隻狗，這隻狗2歲。(第二次提及用 le)

la lune / le soleil / les étoiles 月亮 / 太陽 / 星星 (獨一無二)

✍️ 練習 Exercices

用 son, sa, ses 填空：

1. le père de Marie → ___ père 答案：Son père
2. la mère de Paul → ___ mère 答案：Sa mère
3. la voiture de Jean → ___ voiture 答案：Sa voiture

📚 文法 Grammaire — 疑問詞 quel

quel 是形容詞，意思是「什麼/哪一個」(what / which)，需根據所修飾名詞的陰陽性及單複數做變化。

單數 複數
陽性 quel quels
陰性 quelle quelles

📌 常見句型：

  • Quel/Quelle est + n. ？ → 這個...是什麼？
  • Quels/Quelles sont + n.（複數）？ → 這些...是什麼？

💡 quel 的用法例句

Quel est votre nom? 您的姓氏是什麼？（nom 陽性單數）
Quelle est ta nationalité? 你的國籍是什麼？（nationalité 陰性單數）
Quelles sont ses études? 他/她的專業是什麼？（études 陰性複數）
Quel est notre numéro de téléphone? 我們的電話號碼是什麼？（numéro 陽性單數）
Quelles sont vos coordonnées? 您的聯繫方式是什麼？（coordonnées 陰性複數）

⚠️ 注意：法文感受用 avoir（有），不用 être（是）！
✅ J'ai chaud. (我熱。)  ❌ Je suis chaud.
✅ Tu as faim? (你餓嗎？)  ❌ Tu es faim?

💬 對話練習

— Tu as soif? 你渴嗎？ — Non, j'ai faim. 不，我餓。
— Vous avez sommeil? 你們想睡嗎？ — Oui, nous avons sommeil. 是的，我們想睡。
Elle a douze ans, elle est petite, elle a sommeil. 她12歲，她很小，她很困。
